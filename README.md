# SkillQuest 🎮

SkillQuest is a gamified productivity and habit-tracking web app built with React.js.

The app transforms daily tasks into quests where users earn XP, level up, maintain streaks, and track progress in a game-like interface.

## ✨ Features

- 🎯 Quest-based task system
- ⚡ XP and leveling mechanics
- 🔥 Daily streak tracking
- 💾 Persistent progress using localStorage
- ✅ Quest completion locking
- 📈 Animated XP progress bar
- 🎨 Dynamic UI feedback for completed quests
- 🔄 Daily reset system

## 🛠️ Tech Stack

- React.js
- JavaScript
- CSS
- localStorage API
